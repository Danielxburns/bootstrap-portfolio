Cast and Crew Carousel

Movieoke (aka Fandangit) - A searchable movie app providing trailers, theatres, showtimes, synopsis, and cast info

  • Built using React, Node, Express and MongoDB
  • Responsible for the Cast and Crew feature
  • Worked closely with other engineers using Agile methodologies, performing daily stand-ups and regular code reviews through Git version control
  • Used React with styled-components to create a photo carousel from scratch with wrap-around content that displayed actor’s bio and filmography in a popup modal
  • Integrated other team member's features into a Node based proxy server
  • Deployed onto AWS using Elastic Beanstalk to streamline the deployment process and stored the photo assets in an s3 bucket in order to keep the files available, secure and easy to manage

Fibonacci Calculator

Fibonacci Calculator - a Kubernetes-based application that calculates Fibonacci values

  • Built out a React front end that uses Axios to make requests to an Express server
  • Created a Redis client to calculate the values and cache the results
  • Connected a Postgres database to store input values long term
  • Configured Dockerfiles to generate containers for each service using Docker-compose
  • Set up CI/CD through Github with TravisCI to automate testing and deployment
  • Kubernetes
  • Google Cloud
  • HTTPS certs using Let'sEncrypt!

Back End Scaling Project

BookShelf - A searchable book finding app providing description, synopsis, reviews, ratings, and author details

  • Re-designed the back end of a legacy code application built using React, Node, Express and MySQL to be able to handle 10M primary users
  • Authored NodeJS scripts for generating 100M fake records of book reviews for testing (≈10 reviews/user)
  • Designed schemas for both MongoDB and PosrtgreSQL
  • Performed benchmark testing using identical queries to verify permformance on large data sets and chose PostgreSQL due to a 24% faster response time
  • Deployed application to AWS on multiple EC2 instances and migrated the database
  • Conducted real-time performance testing using New Relic and reduced latency by vertically scaling database instances

Integrated circuit layout

IC layout is the representation of an integrated circuit in terms of planar geometric shapes which correspond to the patterns of metal, oxide, or semiconductor layers that make up the components of the integrated circuit.

These are some of the IC components that I created using a program called Cadence Virtuoso as part of my Electronic Graphics Computer-Aided Design curriculum at ACC. Learning about how these components come together to generate the logic that defines our technological world has been an essential stepping stone on my way to discovering my passion for software engineering as well as sparking my interest in IoT technology.
